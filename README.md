## Features

- 3D 글래스모피즘 카드 (Aurora Border + Holographic Refraction + Spotlight Follow)
- 마우스 추적 3D 틸트 + idle float 애니메이션
- 3층 별빛 패럴랙스 + 성운 배경 + 원근 그리드
- 다크/라이트 모드 슬라이드 토글
- 방문자 카운터 (카운트업 애니메이션)
- 방명록 (localStorage)
- 반응형 디자인 (데스크톱/태블릿/모바일)
- prefers-reduced-motion 지원

## Tech Stack

HTML / CSS / Vanilla JS — 프레임워크 없음
